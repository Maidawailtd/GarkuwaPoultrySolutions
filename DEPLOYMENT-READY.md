1. Go to https://netlify.com
2. Drag the entire `/dist/public/` folder to the deployment area
3. Get instant free hosting with SSL

#### Option 2: GitHub Pages
1. Push your code to GitHub
2. Go to repository Settings > Pages
3. Select source and deploy

#### Option 3: Vercel
1. Go to https://vercel.com
2. Import your project or drag & drop the `/dist/public/` folder
3. Get instant deployment

#### Option 4: Firebase Hosting
1. Go to https://firebase.google.com
2. Create project and enable hosting
3. Upload your `/dist/public/` folder
