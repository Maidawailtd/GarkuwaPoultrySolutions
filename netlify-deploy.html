Free Deployment Options for Garkuwa Poultry Farm

Option 1: Netlify (Recommended)

Step 1: Build your site

npm run build

Step 2: Deploy

1. Go to netlify.com

2. Drag and drop your dist/client folder

3. Get instant free hosting with SSL

Option 2: Vercel

npm install -g vercel
vercel login
vercel --prod

Option 3: Surge.sh

npm install -g surge
npm run build
cd dist/client
surge

Option 4: GitHub Pages

1. Push your code to GitHub

2. Enable Pages in repository settings

3. Select source as GitHub Actions

Shell Commands for Quick Deployment:

# Build the project
npm run build

# Option A: Use Netlify CLI
npm install -g netlify-cli
netlify deploy --prod --dir=dist/client

# Option B: Use Surge
npm install -g surge
cd dist/client && surge

# Option C: Use Vercel
npm install -g vercel
vercel --prod
